# Halfwidth-Forms-Converter

## Feature

将全角符号转换为半角符号

| From | To  |
| ---- | --- |
| ，   | ,   |
| 。   | .   |
| ：   | :   |
| ？   | ?   |
| ；   | ;   |
| 、   | ,   |
| （   | (   |
| ）   | )   |
