# Halfwidth-Forms-Converter

## Feature

将全角符号转换为半角符号, 并在中英文之间自动加入空格 (pangu 实现)

| From | To  |
| ---- | --- |
| ,   | ,   |
| .   | .   |
| :   | :   |
| ?   | ?   |
| ;   | ;   |
| ,   | ,   |
| (   | (   |
| )   | )   |

## Usage

`⌘ ⇧ P` / `F1` 调出命令面板, 键入 `Half Width Forms Convert`, 选择使用即可.
